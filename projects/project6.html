Chit-Chat

The objective was to build a React-Native chat application for iOS and Android, utilising Google Firebase storage and with an ability to share images and location.

Users can join the chat group, express preferences for background and dispaly name, and share images directly from the device camera or picture gallery.

Offline function is preserved through the use of local or async storage.

Construction Tools

  • React-Native framework
  • Expo build tools
  • Gifted Chat library
  • Google Firebase Storage and Cloud Firestore
  • Built and tested for both iOS and Android
  • Trello project-managed
